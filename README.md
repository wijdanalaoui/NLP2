This project implements deep learning models to perform binary sentiment analysis on text data, classifying reviews as positive or negative. Two architectures were used: a Long Short-Term Memory (LSTM) network and a one-dimensional Convolutional Neural Network (1D CNN). The LSTM model captures long-term dependencies in sequences by processing text step by step through an embedding layer followed by an LSTM layer and a sigmoid output layer. In contrast, the CNN model extracts local patterns using convolutional and pooling layers after the embedding stage, making it faster while still effective at identifying key textual features.

The data was preprocessed through tokenization and padding to ensure consistent input length, and labels were encoded numerically. The dataset was split into training, validation, and test sets using stratified sampling to preserve class distribution. Word embeddings were learned through an embedding layer, enabling the models to capture semantic relationships between words.

Both models were trained using the binary cross-entropy loss function and optimized with the Adam optimizer, which adapts learning rates for efficient and stable convergence. Training was performed using mini-batch gradient descent with backpropagation, and validation data was used to monitor performance and reduce overfitting. Evaluation was conducted using accuracy, confusion matrix, and classification metrics such as precision, recall, and F1-score.

Overall, the LSTM model is effective for capturing contextual information but is computationally heavier, while the CNN model offers faster training and strong performance in detecting important patterns.
